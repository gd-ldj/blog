## JS-Code-Specification
> 本规范基于ES6

### 编码风格
 + 缩进, 和html缩进保持一致，2或4个空格；
 + 分号，统一以分号结束语句，可以避免JS引擎自动插入机制的怪异行为，语义上也更加明了；
 + 逗号，对于逗号分隔的多行结构，不使用行首逗号；并且始终加上最后一个逗号；
